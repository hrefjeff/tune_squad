The University of North Alabama has to schedule 227 exams in x exam periods
so that there are no exam conflicts.

G = (N, A)

N = {1, 2, 3, … n} set of exams. 28 periods.

A = set of arcs.

(i, j) ∈ A if a person needs to take exam i and exam j.
